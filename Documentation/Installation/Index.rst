============
Installation
============

Using composer
==============

Use `composer require jweiland/pfprojects` inside your TYPO3 project to get the extension.
Then activate it using the TYPO3 Extension Manager.

Using TER
=========

Open the Extension Manager in the backend, select "Get Extensions", update the extension list
and search for pfprojects. Then press "Import and install".
